- ✅ Calculating Percentage
- ✅ Factorial of a number
- ✅ Finding LCM

- ✅ Palindrome check
- ✅ Fibonacci series
- ✅ Pattern printing
- ✅ Simple recursion problems

These programs are useful for beginners who are just starting with coding.

## 🚀 Languages Used

- Python 🐍
- C / C++
